Haiku.lt est construit selon cette philosophie :

- Les outils pour les professionnels indépendants devraient être :

  - **Simples** - Haiku.lt se concentre sur les fonctionnalités essentielles sans superflu inutile.

  - **Rapides** - Haiku.lt est conçu pour minimiser le temps que vous passez sur les tâches de facturation.

  - **Flexibles** - Haiku.lt évite les limitations artificielles qui vous empêcheraient d'accomplir vos tâches.

  - **Préventifs des erreurs** - La flexibilité peut créer des opportunités d'erreurs, donc Haiku.lt vous aide à éviter les erreurs courantes dans la mesure du possible.

  - **Accessibles partout** - Haiku.lt fonctionne aussi bien sur les ordinateurs de bureau que sur les appareils mobiles.

- Le plan gratuit n'aura jamais moins de fonctionnalités qu'il n'en a actuellement, sauf si des circonstances indépendantes de notre volonté augmentent le coût de maintien du niveau gratuit. Dans tous les cas, nous vous informerons de tout changement.

- Le plan payant offre ces avantages :

  - Plus de fonctionnalités vous permettant de travailler plus rapidement et d'accomplir davantage

  - Support prioritaire lorsque vous avez des questions ou des problèmes

  - Prise en compte de vos demandes de fonctionnalités
