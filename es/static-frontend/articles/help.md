Haiku.lt está diseñado para ayudarte a crear facturas de manera rápida y fácil, y gestionar tu flujo de trabajo de facturación.

Si tienes curiosidad sobre por qué deberías elegir Haiku.lt, echa un vistazo a nuestra [filosofía](/es/philosophy).

También puedes revisar el [registro de cambios](/es/changelog) para ver las últimas funciones y mejoras.

## Gestión de facturas

El sistema proporciona herramientas completas para gestionar tus facturas:

- [Creación y edición de facturas](/es/invoice-management)

  Aprende a crear facturas con números de serie adecuados, añadir información del comprador y vendedor, gestionar artículos de línea y trabajar con diferentes tipos de facturas, incluidas facturas proforma y de crédito.

- [Envío de facturas](/es/invoice-sending)

  Descubre cómo enviar facturas directamente desde Haiku.lt usando la integración con Gmail, personalizar plantillas de correo electrónico y usar variables para personalizar tus correos de facturas.

- [Programación de información adicional](/es/extra-info-programming)

  Función avanzada para programadores: rellena automáticamente el campo de información adicional según los datos de la factura usando JavaScript.
