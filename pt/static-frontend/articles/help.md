O Haiku.lt foi desenhado para ajudá-lo a criar faturas de forma rápida e fácil e a gerir o seu fluxo de trabalho de faturação.

Se tem curiosidade sobre por que deve escolher o Haiku.lt, consulte a nossa [filosofia](/pt/philosophy).

Também pode rever o [registo de alterações](/pt/changelog) para ver as funcionalidades e melhorias mais recentes.

## Gestão de Faturas

O sistema fornece ferramentas abrangentes para gerir as suas faturas:

- [Criar e editar faturas](/pt/invoice-management)

  Aprenda como criar faturas com números de série adequados, adicionar informação de compradores e vendedores, gerir itens de linha e trabalhar com diferentes tipos de fatura, incluindo faturas proforma e de crédito.

- [Enviar faturas](/pt/invoice-sending)

  Descubra como enviar faturas diretamente do Haiku.lt usando integração com Gmail, personalizar modelos de email e usar variáveis para personalizar os seus emails de faturas.

- [Programação de informação adicional](/pt/extra-info-programming)

  Funcionalidade avançada para programadores: preencher automaticamente o campo de informação adicional com base nos dados da fatura usando JavaScript.
