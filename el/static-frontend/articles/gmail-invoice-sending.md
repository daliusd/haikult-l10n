Όταν στέλνετε ένα τιμολόγιο σε έναν πελάτη, η διεύθυνση ηλεκτρονικού ταχυδρομείου που βλέπουν έχει μεγαλύτερη σημασία από ό,τι μπορεί να φανταστείτε. Ένα μήνυμα από "noreply@invoicingapp.com" ή "invoices@random-service.com" δεν εμπνέει εμπιστοσύνη. Φαίνεται αυτοματοποιημένο, απρόσωπο και, για να είμαι ειλικρινής, λίγο ύποπτο. Οι πελάτες είναι πιο πιθανό να το αγνοήσουν, να το στείλουν στον φάκελο ανεπιθύμητης αλληλογραφίας ή να διστάζουν πριν ανοίξουν το συνημμένο αρχείο.

Τι γίνεται αν τα τιμολόγιά σας μπορούσαν να φτάσουν από την πραγματική διεύθυνση ηλεκτρονικού ταχυδρομείου της επιχείρησής σας; Η καλή είδηση είναι ότι μπορούν, και είναι ευκολότερο από ό,τι νομίζετε. Αυτός ο οδηγός θα σας δείξει πώς να στέλνετε επαγγελματικά τιμολόγια απευθείας από το λογαριασμό Gmail σας, να συγκρίνετε τις διαθέσιμες τεχνικές μεθόδους και να εξηγήσετε γιατί αυτό είναι σημαντικό για την επιχείρησή σας.

## Το Πρόβλημα με τα Γενικά Τιμολόγια Ηλεκτρονικού Ταχυδρομείου

Οι περισσότερες υπηρεσίες ηλεκτρονικής τιμολόγησης στέλνουν μηνύματα εξ ονόματός σας χρησιμοποιώντας τις δικές τους διευθύνσεις ηλεκτρονικού ταχυδρομείου ή γενικές διευθύνσεις "noreply". Αν και αυτό μπορεί να φαίνεται βολικό, δημιουργεί αρκετά προβλήματα:

**Ζητήματα εμπιστοσύνης:** Όταν οι πελάτες λαμβάνουν ένα τιμολόγιο από μια άγνωστη διεύθυνση αποστολέα, φυσικά γίνονται προσεκτικοί. Είναι αυτό νόμιμο; Ο freelancer μου έστειλε πραγματικά αυτό; Αυτές οι ερωτήσεις δεν θα έπρεπε καν να προκύψουν, αλλά οι γενικές διευθύνσεις αποστολέα φυτεύουν σπόρους αμφιβολίας.

**Ζητήματα παράδοσης:** Οι υπηρεσίες ηλεκτρονικού ταχυδρομείου όπως το Gmail και το Outlook γίνονται όλο και πιο αυστηρές στο φιλτράρισμα ανεπιθύμητης αλληλογραφίας. Τα μηνύματα από άγνωστες υπηρεσίες είναι πιο πιθανό να καταλήξουν στους φακέλους ανεπιθύμητης αλληλογραφίας, ειδικά αν ο τομέας του αποστολέα δεν ταιριάζει με την επιχείρησή σας. Το τέλειο επαγγελματικό τιμολόγιό σας μπορεί να μην φανεί ποτέ.

**Έλλειψη προσωποποίησης:** Οι γενικές διευθύνσεις αποστολέα δημιουργούν απόσταση μεταξύ σας και του πελάτη σας. Κάνουν την επικοινωνία της επιχείρησής σας να φαίνεται αυτοματοποιημένη και συναλλακτική, παρά προσωπική και επαγγελματική.

**Εμπόδια απάντησης:** Όταν ένας πελάτης θέλει να κάνει μια ερώτηση σχετικά με ένα τιμολόγιο, θα έπρεπε να μπορεί απλώς να κάνει κλικ "απάντηση". Με διευθύνσεις noreply ή αποστολείς τρίτων, αυτό γίνεται άβολο ή ακόμη και αδύνατο. Οι πελάτες πρέπει να ψάξουν για τα πραγματικά στοιχεία επικοινωνίας σας, προσθέτοντας τριβή σε ό,τι θα έπρεπε να ήταν μια απλή ανταλλαγή.

Ο επιχειρηματικός αντίκτυπος αυτών των ζητημάτων είναι πραγματικός. Οι καθυστερημένες προβολές τιμολογίων σημαίνουν καθυστερημένες πληρωμές. Η σύγχυση σχετικά με τη νομιμότητα του τιμολογίου σημαίνει χαμένο χρόνο σε αρχ-πίσω διευκρινίσεις. Η επαγγελματική φήμη σας αξίζει καλύτερα.

## Gmail API vs SMTP: Δύο Προσεγγίσεις για την Αποστολή Τιμολογίων

Αν θέλετε να στέλνετε μηνύματα ηλεκτρονικού ταχυδρομείου από το δικό σας λογαριασμό Gmail μέσω μιας εφαρμογής, υπάρχουν δύο κύριες τεχνικές προσεγγίσεις: SMTP και Gmail API. Η κατανόηση της διαφοράς βοηθά να εξηγηθεί γιατί η σύγχρονη προσέγγιση είναι τόσο ευκολότερη όσο και ασφαλέστερη.

### Παραδοσιακή Προσέγγιση SMTP

Το SMTP (Simple Mail Transfer Protocol) είναι ο δεκαετών παλιός πρότυπος για την αποστολή ηλεκτρονικού ταχυδρομείου. Πολλές υπηρεσίες το χρησιμοποιούν ακόμα επειδή είναι καθολικό και λειτουργεί με οποιοδήποτε πάροχο ηλεκτρονικού ταχυδρομείου.

**Πώς λειτουργεί:** Δίνετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου και τον κωδικό πρόσβασης (ή έναν κωδικό πρόσβασης ειδικό για την εφαρμογή) στην εφαρμογή ηλεκτρονικής τιμολόγησης. Η εφαρμογή αποθηκεύει αυτά τα διαπιστευτήρια και τα χρησιμοποιεί για την αποστολή μηνυμάτων μέσω του SMTP διακομιστή του Gmail εξ ονόματός σας.

**Πλεονεκτήματα:** Λειτουργεί με οποιοδήποτε πάροχο ηλεκτρονικού ταχυδρομείου, όχι μόνο Gmail. Ευρέως υποστηριζόμενο από παλαιότερες εφαρμογές.

**Μειονεκτήματα:** Λιγότερο ασφαλές επειδή μοιράζεστε διαπιστευτήρια. Απαιτεί τη δημιουργία και διαχείριση κωδικών πρόσβασης ειδικών για την εφαρμογή. Πιο περίπλοκη εγκατάσταση με διευθύνσεις διακομιστή και αριθμούς θυρών. Εάν η υπηρεσία παραβιαστεί, τα διαπιστευτήρια ηλεκτρονικού ταχυδρομείου σας θα μπορούσαν να εκτεθούν.

### Σύγχρονη Προσέγγιση Gmail API

Το Gmail API είναι η σύγχρονη λύση της Google για να αλληλεπιδρούν οι εφαρμογές με το Gmail με ασφάλεια. Αντί να μοιράζεστε τον κωδικό πρόσβασης, εξουσιοδοτείτε συγκεκριμένα δικαιώματα.

**Πώς λειτουργεί:** Όταν εξουσιοδοτείτε μια εφαρμογή να στέλνει μηνύματα, η Google δημιουργεί ένα ασφαλές token που χορηγεί μόνο την άδεια αποστολής ηλεκτρονικού ταχυδρομείου. Δεν μοιράζεστε ποτέ τον κωδικό πρόσβασης. Μπορείτε να ανακαλέσετε αυτήν την πρόσβαση ανά πάσα στιγμή από τις ρυθμίσεις του λογαριασμού σας στο Google.

**Πλεονεκτήματα:** Πολύ πιο ασφαλές (OAuth2 ταυτοποίηση, χωρίς κοινή χρήση κωδικών πρόσβασης). Απλούστερη εγκατάσταση (απλώς κάντε κλικ "εξουσιοδότηση"). Καλύτερος έλεγχος δικαιωμάτων (χορηγεί μόνο αυτό που χρειάζεται). Τα μηνύματα προέρχονται πραγματικά από το λογαριασμό Gmail σας. Μπορείτε να ανακαλέσετε την πρόσβαση σε δευτερόλεπτα εάν χρειαστεί.

**Μειονεκτήματα:** Λειτουργεί μόνο με Gmail (απαιτείται λογαριασμός Google).

### Γρήγορη Σύγκριση

- **Ασφάλεια:** Το Gmail API κερδίζει αποφασιστικά. Η ταυτοποίηση OAuth2 είναι πολύ πιο ασφαλής από την αποθήκευση διαπιστευτηρίων ηλεκτρονικού ταχυδρομείου.
- **Πολυπλοκότητα εγκατάστασης:** Το Gmail API είναι πιο απλό. Ένα κλικ για εξουσιοδότηση έναντι της διαμόρφωσης ρυθμίσεων διακομιστή και της δημιουργίας ειδικών κωδικών πρόσβασης.
- **Διεύθυνση αποστολέα:** Το Gmail API στέλνει από τη πραγματική σας διεύθυνση Gmail. Το SMTP μπορεί επίσης, αλλά η διαμόρφωση είναι πιο περίπλοκη.
- **Παράδοση:** Το Gmail API επωφελείται από την πλήρη υποδομή ταυτοποίησης του Gmail (SPF, DKIM, DMARC).

Για freelancers και μικρές επιχειρήσεις που χρησιμοποιούν Gmail, η προσέγγιση API είναι σαφώς ανώτερη. Είναι ευκολότερη, ασφαλέστερη και παρέχει καλύτερα αποτελέσματα.

## Πώς το Haiku.lt Χρησιμοποιεί Gmail API για Επαγγελματική Αποστολή Τιμολογίων

Το Haiku.lt αξιοποιεί το Gmail API για να διασφαλίσει ότι τα τιμολόγιά σας φτάνουν από τη πραγματική σας διεύθυνση ηλεκτρονικού ταχυδρομείου, όχι από κάποια γενική διεύθυνση υπηρεσίας.

Ενδεικτικά αυτά που συμβαίνουν στο παρασκήνιο:

Όταν συνδέεστε στο Haiku.lt με το λογαριασμό σας Google, σας ζητείται να δώσετε άδεια στην εφαρμογή να στέλνει μηνύματα εξ ονόματός σας. Αυτό χρησιμοποιεί την εξουσιοδότηση OAuth2 της Google, που είναι το ίδιο ασφαλές σύστημα που χρησιμοποιούν αξιόπιστες εφαρμογές σε όλο το διαδίκτυο.

Μόλις εξουσιοδοτηθεί, το Haiku.lt μπορεί να στέλνει τιμολόγια απευθείας μέσω του λογαριασμού Gmail σας. Η βασική διαφορά από άλλες υπηρεσίες: το μήνυμα προέρχεται πραγματικά από τη διεύθυνση Gmail σας. Οι πελάτες σας βλέπουν τη πραγματική σας διεύθυνση ηλεκτρονικού ταχυδρομείου στα εισερχόμενά τους. Η φήμη του τομέα σας διατηρείται. Η υποδομή ταυτοποίησης Gmail (SPF, DKIM, DMARC) λειτουργεί τέλεια επειδή το μήνυμα προέρχεται πραγματικά από το λογαριασμό Gmail σας.

Κανένας κωδικός πρόσβασης δεν αποθηκεύεται ποτέ. Δεν απαιτείται περίπλοκη διαμόρφωση. Και μπορείτε να ανακαλέσετε την πρόσβαση του Haiku.lt ανά πάσα στιγμή μέσω των ρυθμίσεων του λογαριασμού σας στο Google, εάν χρειαστεί.

Όταν ο πελάτης σας λαμβάνει το τιμολόγιό σας, βλέπει τη διεύθυνση ηλεκτρονικού ταχυδρομείου σας και το όνομά σας. Φαίνεται σαν να του στείλατε ένα μήνυμα απευθείας - γιατί ουσιαστικά το κάνατε. Αυτή η μικρή λεπτομέρεια δημιουργεί έκπληξη, αλλά έχει μεγάλη διαφορά στο πώς τα τιμολόγιά σας αντιλαμβάνονται και χειρίζονται.

## Οδηγός Εγκατάστασης Βήμα προς Βήμα

Η ρύθμιση της επαγγελματικής αποστολής τιμολογίων με το λογαριασμό Gmail σας στο Haiku.lt διαρκεί μόνο λίγα λεπτά.

### Βήμα 1: Εξουσιοδότηση Πρόσβασης Gmail

Όταν συνδέεστε στο Haiku.lt για πρώτη φορά, θα σας ζητηθεί να συνδεθείτε με το λογαριασμό σας Google. Κατά τη διάρκεια αυτής της διαδικασίας, η Google θα σας δείξει τα δικαιώματα που ζητάει το Haiku.lt, συμπεριλαμβανομένης της δυνατότητας αποστολής μηνυμάτων εξ ονόματός σας.

Ελέγξτε τα δικαιώματα και κάντε κλικ "Αποδοχή" για να χορηγήσετε πρόσβαση. Αυτή είναι μία εξουσιοδότηση. Εάν αρχικά συνδεθήκατε χωρίς να χορηγήσετε δικαίωμα ηλεκτρονικού ταχυδρομείου, μπορείτε να αποσυνδεθείτε και να συνδεθείτε ξανά για να προσθέσετε αυτό το δικαίωμα.

### Βήμα 2: Διαμόρφωση Ρυθμίσεων Ηλεκτρονικού Ταχυδρομείου

Μόλις εξουσιοδοτήσετε την πρόσβαση ηλεκτρονικού ταχυδρομείου, πραγματοποιήστε πλοήγηση στη σελίδα [Ρυθμίσεις](/app/settings) στο Haiku.lt. Εδώ μπορείτε να προσαρμόσετε το εμφάνιση των τιμολογίων ηλεκτρονικού ταχυδρομείου σας και το περιεχόμενό τους.

**Προσαρμόστε το θέμα του μηνύματος:** Μπορείτε να δημιουργήσετε δυναμικά θέματα μηνυμάτων χρησιμοποιώντας μεταβλητές όπως `{{invoiceNo}}` για τον αριθμό τιμολογίου, `{{buyer}}` για το όνομα του αγοραστή, `{{price}}` για το ποσό του τιμολογίου και `{{invoiceDate}}` για την ημερομηνία. Για παράδειγμα: "Τιμολόγιο {{invoiceNo}} από {{seller}} - {{price}}"

**Επιλέξτε ένα πρότυπο ηλεκτρονικού ταχυδρομείου:** Το Haiku.lt προσφέρει πέντε ενσωματωμένα πρότυπα:
- **Απλό κείμενο** - Απλή, καθολικά συμβατή μορφή κειμένου
- **Απλό HTML** - Όμορφα διαμορφωμένο μήνυμα HTML
- **Με λογότυπο** - Πρότυπο HTML με το λογότυπο της εταιρείας σας
- **Με λογότυπο και τιμή** - Εμφανίζει το λογότυπό σας και το ποσό του τιμολογίου
- **Με λογότυπο, τιμή και πρόσθετες πληροφορίες** - Εμφανίζει πλήρεις πληροφορίες τιμολογίου

Για προχωρημένους χρήστες, μπορείτε να δημιουργήσετε προσαρμοσμένα πρότυπα MJML για πλήρη έλεγχο του σχεδιασμού του ηλεκτρονικού ταχυδρομείου.

**Μεταφορτώστε το λογότυπό σας:** Προσθέστε το λογότυπο της εταιρείας σας για να κάνετε τα ηλεκτρονικά ταχυδρομεία με σήμα ακόμη πιο επαγγελματικά.

**Ορίστε χρώματα σήματος:** Προσαρμόστε τα χρώματα που χρησιμοποιούνται στα πρότυπα ηλεκτρονικού ταχυδρομείου HTML για να ταιριάζουν με την ταυτότητα του σήματος σας.

### Βήμα 3: Στείλτε το Πρώτο Τιμολόγιό σας

Η δημιουργία και η αποστολή ενός τιμολογίου είναι απλή:

1. Δημιουργήστε το τιμολόγιό σας με όλες τις απαραίτητες λεπτομέρειες (αγοραστής, πωλητής, στοιχεία γραμμής κ.λπ.)
2. Εισάγετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του αγοραστή στη φόρμα τιμολογίου
3. Κάντε κλικ στο κουμπί "Αποστολή τιμολογίου"
4. Το τιμολόγιο κλειδώνεται αυτόματα (αποτρέποντας περαιτέρω επεξεργασίες) και αποστέλλεται αμέσως από το λογαριασμό Gmail σας

Ο πελάτης σας λαμβάνει ένα μήνυμα από τη πραγματική σας διεύθυνση Gmail με το PDF του τιμολογίου συνημμένο. Μπορείτε να βρείτε λεπτομερείς πληροφορίες σχετικά με τη διαδικασία αποστολής στον οδηγό [Αποστολή Τιμολογίων](/el/invoice-sending) μας.

## Πλεονεκτήματα Παράδοσης από τον Λογαριασμό Gmail σας

Η χρήση της πραγματικής σας διεύθυνσης Gmail για την αποστολή τιμολογίων παρέχει σημαντικά πλεονεκτήματα παράδοσης σε σχέση με τις γενικές διευθύνσεις υπηρεσίας.

**Καλύτερη τοποθέτηση στα εισερχόμενα:** Οι υπηρεσίες ηλεκτρονικού ταχυδρομείου εμπιστεύονται πολύ περισσότερο τους καθιερωμένους λογαριασμούς Gmail από ό,τι τις άγνωστες υπηρεσίες τρίτων. Τα τιμολόγιά σας είναι πολύ πιο πιθανό να καταλήξουν στα κύρια εισερχόμενα παρά στους φακέλους ανεπιθύμητης αλληλογραφίας ή διαφημίσεων.

**Υποδομή ταυτοποίησης Gmail:** Όταν στέλνετε από το λογαριασμό Gmail σας, όλοι οι μηχανισμοί ταυτοποίησης του Gmail (SPF, DKIM, DMARC) λειτουργούν τέλεια. Αυτά τα τεχνικά πρότυπα λένε στους αποδεκτικούς διακομιστές ηλεκτρονικού ταχυδρομείου ότι το μήνυμά σας είναι νόμιμο και δεν έχει παραμορφωθεί.

**Αναγνώριση παραλήπτη:** Οι πελάτες σας γνωρίζουν ήδη τη διεύθυνση ηλεκτρονικού ταχυδρομείου σας. Όταν το βλέπουν στα εισερχόμενά τους, το αναγνωρίζουν αμέσως ως νόμιμο. Δεν υπάρχει διστασμός, καμία δεύτερη σκέψη και χωρίς ανάγκη να επαληθευτεί ο αποστολέας.

**Εύκολες απαντήσεις:** Εάν ο πελάτης σας έχει ερωτήσεις σχετικά με ένα τιμολόγιο, μπορεί απλώς να κάνει κλικ απάντηση. Η συζήτηση παραμένει στο κανονικό νήμα ηλεκτρονικού ταχυδρομείου μαζί σας, καθιστώντας την επικοινωνία απρόσκοπτη και φυσική.

**Φήμη αποστολέα:** Ο λογαριασμός Gmail σας δημιουργεί φήμη με την πάροδο του χρόνου. Η αποστολή τιμολογίων από το λογαριασμό σας διατηρεί και ενισχύει αυτήν τη φήμη, βελτιώνοντας την παράδοση για όλα τα μηνύματα ηλεκτρονικού ταχυδρομείου της επιχείρησής σας.

**Υψηλότερα ποσοστά ανοίγματος:** Όταν οι πελάτες αναγνωρίζουν και εμπιστεύονται τον αποστολέα, ανοίγουν τα μηνύματα πιο γρήγορα. Αυτό μεταφράζεται απευθείας σε ταχύτερη ανασκόπηση τιμολογίου και τελικά ταχύτερη πληρωμή.

## Επιλογές Προσαρμογής Ηλεκτρονικού Ταχυδρομείου

Πέρα από το απλό στέλνειν από το λογαριασμό Gmail σας, το Haiku.lt προσφέρει εκτεταμένες επιλογές προσαρμογής για να κάνετε τα τιμολόγιά ηλεκτρονικού ταχυδρομείου σας να ταιριάζουν με το σήμα και το στυλ επικοινωνίας σας.

Μπορείτε να επιλέξετε από πολλά πρότυπα που κυμαίνονται από απλό κείμενο έως όμορφα διαμορφωμένα μηνύματα HTML με το λογότυπό σας, τα χρώματα του σήματος και τις λεπτομέρειες του τιμολογίου. Τα πρότυπα χρησιμοποιούν τεχνολογία MJML, η οποία διασφαλίζει ότι φαίνονται φανταστικά σε όλους τους κλάδους ηλεκτρονικού ταχυδρομείου - από το Gmail έως το Outlook έως τις εφαρμογές κινητών ηλεκτρονικού ταχυδρομείου.

Οι δυναμικές μεταβλητές σας επιτρέπουν να προσαρμόσετε προσωπικά κάθε μήνυμα αυτόματα. Συμπεριλάβετε τον αριθμό τιμολογίου, το όνομα του αγοραστή, το συνολικό ποσό, την ημερομηνία του τιμολογίου και άλλες λεπτομέρειες χωρίς να χρειάζεται να τα πληκτρολογήσετε χειροκίνητα για κάθε τιμολόγιο. Το σύστημα συμπληρώνει τις μεταβλητές αυτόματα με βάση τα δεδομένα του τιμολογίου σας.

Για προχωρημένους χρήστες που θέλουν πλήρη έλεγχο, υποστηρίζονται προσαρμοσμένα πρότυπα MJML. Μπορείτε να δημιουργήσετε πρότυπα που ταιριάζουν τέλεια με τις οδηγίες του σήματος σας. Εάν δεν είστε εξοικειωμένοι με το MJML, μπορείτε ακόμη και να ζητήσετε από εργαλεία τεχνητής νοημοσύνης να σας βοηθήσουν να δημιουργήσετε πρότυπα με βάση την περιγραφή σας σχετικά με το πώς θέλετε το μήνυμα να φαίνεται.

Όλη αυτή η προσαρμογή συνδυάζεται με την επαγγελματική παράδοση από το λογαριασμό Gmail σας για να δημιουργήσει τιμολόγια ηλεκτρονικού ταχυδρομείου που αντιπροσωπεύουν την επιχείρησή σας ακριβώς όπως θέλετε.

## Ξεκινήστε να Στέλνετε Επαγγελματικά Τιμολόγια Σήμερα

Η αποστολή τιμολογίων από τον πραγματικό σας λογαριασμό Gmail, αντί από μια γενική διεύθυνση υπηρεσίας, είναι μια μικρή αλλαγή που κάνει μεγάλη διαφορά. Οι πελάτες σας βλέπουν μια γνώριμη, αξιόπιστη διεύθυνση ηλεκτρονικού ταχυδρομείου. Τα τιμολόγιά σας αποφεύγουν τους φακέλους ανεπιθύμητης αλληλογραφίας. Η επικοινωνία της επιχείρησής σας αισθάνεται πιο προσωπική και επαγγελματική.

Με το Haiku.lt, αυτή η επαγγελματική παράδοση τιμολογίου είναι διαθέσιμη τόσο στα δωρεάν όσο και στα Pro σχέδια. Το δωρεάν σχέδιο περιλαμβάνει 500 τιμολόγια με πλήρεις δυνατότητες αποστολής ηλεκτρονικού ταχυδρομείου - αρκετό για τους περισσότερους freelancers να το χρησιμοποιήσουν για χρόνια. Εάν χρειάζεστε απεριόριστα τιμολόγια και πρόσθετες επιλογές προσαρμογής, το Pro σχέδιο κοστίζει μόνο €5 ανά μήνα ή €48 ανά έτος.

Η ρύθμιση διαρκεί μόνο λίγα λεπτά: εξουσιοδοτήστε την πρόσβαση Gmail, προσαρμόστε το πρότυπο ηλεκτρονικού ταχυδρομείου σας και ξεκινήστε να στέλνετε επαγγελματικά τιμολόγια από τη δικη σας διεύθυνση ηλεκτρονικού ταχυδρομείου. Χωρίς περίπλοκη διαμόρφωση SMTP. Χωρίς αποθηκευμένους κωδικούς πρόσβασης. Χωρίς γενικές διευθύνσεις αποστολέα.

Τα τιμολόγιά σας αξίζουν να φαίνονται επαγγελματικά από τον αποστολέα έως την υπογραφή. Ξεκινήστε να τα στέλνετε από το λογαριασμό Gmail σας σήμερα στο [haiku.lt](https://haiku.lt).

Για περισσότερες πληροφορίες, δείτε τον λεπτομερή οδηγό [Αποστολή Τιμολογίων](/el/invoice-sending) μας ή επισκεφτείτε τη σελίδα [Βοήθεια](/el/help) μας.
